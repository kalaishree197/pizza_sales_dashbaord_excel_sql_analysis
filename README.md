# 🍕 Pizza Sales Dashboard - Excel & SQL Analysis

*Interactive Excel Dashboard with Supporting SQL Queries*

A practical data analytics project analyzing pizza sales data using SQL for calculations and Excel for visualization.

---

## 📋 Project Overview
This project demonstrates end-to-end data analysis:
- SQL queries to calculate KPIs and trends
- An interactive Excel dashboard built with tables, charts, and timeline

The project is based on a YouTube tutorial. I followed the steps, implemented the queries, built the dashboard, and added my own insights.

---

## 🛠 Tech Stack
- *SQL* – KPI calculations, trends, and aggregations
- *Microsoft Excel* – Tables, Charts, Pivot Tables, and Timeline

## 📊 Key KPIs
- Total Revenue: *$68,737*
- Average Order Value: *$38.21*
- Total Pizzas Sold: *4,151*
- Total Orders: *1,799*

---

## 🧠 Key Business Insights
- Large pizzas contribute *44.55%* of total revenue
- Classic category leads in sales
- Peak ordering hours are between 12–1 PM and after 4–8 PM
- Wednesday and Thursday are the busiest days
